## Installation

### Prerequisites

- npm

- NodeJs

- VSCode

### Installing and launching

Clone the repository of Kasa:

`git clone https://github.com/JM97142/Kasa_React`

Install dependencies:

`npm install`

Lauch Front-End:

`npm start`

Front-End launch at URL `http://localhost:3000`
